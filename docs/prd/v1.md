# ESPN NY Content Studio

## Overview

ESPN NY Content Studio is a cloud-based content creation platform for ESPN New York producers.

The platform allows users to create live show thumbnails, generate SEO-optimized YouTube titles, generate descriptions, manage assets, and collaborate on projects.

## Goals

- Create thumbnails quickly
- Maintain ESPN NY branding
- Reduce repetitive design work
- Generate SEO content
- Support future content modes

## V1 Scope

- Live Show Thumbnail Mode
- AI Headlines
- AI SEO Titles
- AI Descriptions
- Asset Management
- Project Sharing
- Version History

## Future Scope

- Segment Thumbnails
- Social Media Graphics
- Breaking News Mode
